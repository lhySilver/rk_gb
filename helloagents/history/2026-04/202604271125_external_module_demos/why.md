# 变更提案: 外部模块接入 Demo 文档

## 背景
协议配置和 GAT1400 上报接口已经收口到 `ProtocolManager` 与 `LowerGAT1400SDK`，但外部模块开发人员需要快速知道“该调哪个接口、字段怎么填、什么时候需要重启服务”。

## 目标
- 新增一份面向外部模块开发的文档型 demo。
- 覆盖 GB 标准注册配置、GB 零配置、GAT1400 注册配置、在线状态查询和 1400 结构化对象上报。
- 不新增示例编译目标，不改 Makefile/CMake。

## 非目标
- 不新增运行时代码。
- 不新增外部模块适配层。
- 不替调用方封装业务字段填充逻辑。

## 验收标准
- 文档包含可复制的最小 C++ 片段。
- 文档明确 `Set*` 只写 flash，运行中生效需要显式 `Restart*RegisterService()`。
- 现有 API/模块文档能链接到 demo 文档。
- 文档回归和 `git diff --check` 通过。
